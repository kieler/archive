This application represents the J2ME (Java Minimal Edition) remote
client part that can be run on J2ME supporting mobile phones.

You will need the de.cau.cs.kieler.sim.mobile.tablemaster plug-in
running together with the KIELER Execution Manager 
(de.cau.cs.kieler.sim.kiem) in order to setup a connection.
